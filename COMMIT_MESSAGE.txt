Рефакторинг и расширение функциональности ML сервиса

Основные изменения:

Backend:
- Добавлена система событий (event_emitter) для отслеживания и вещания событий
- Реализован планировщик задач (scheduler) с поддержкой приоритетов
- Добавлен пул воркеров (worker_pool) для управления параллельными задачами
- Реализована очередь с приоритетами (priority_queue)
- Обновлены API endpoints для поддержки новой функциональности
- Улучшена система безопасности и аутентификации
- Обновлены модели базы данных и репозитории

Frontend:
- Добавлены новые компоненты для отображения событий (EventsTab, EventCard, EventTile)
- Добавлены компоненты для управления заданиями (JobsTab, JobCard)
- Реализованы табы для моделей, обучения и предсказаний (ModelsTab, TrainingTab, PredictTab)
- Добавлен компонент обзора (OverviewTab)
- Улучшен UI/UX с новыми стилями и анимациями
- Обновлена интеграция с API и WebSocket

Инфраструктура:
- Обновлен .gitignore для исключения временных файлов и артефактов
- Удалена устаревшая документация и скрипты настройки
- Улучшена структура проекта
